= ScanDB

* http://scandb.rubyforge.org/
* Postmodern Modulus III (postmodern.mod3 at gmail.com)

== DESCRIPTION:

ScanDB is a library for importing and analyzing information generated by
various network scanning utilities.

== FEATURES/PROBLEMS:

* Imports scan information from Nmap XML scan files.

== REQUIREMENTS:

* DataMapper:
  * do_sqlite3 >= 0.9.2
  * dm-core >= 0.9.2
  * dm-types >= 0.9.2
* libxml-ruby

== INSTALL:

  $ sudo gem install scandb
